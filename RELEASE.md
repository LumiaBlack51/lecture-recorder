# Release Notes

## Assets
- `lecture-recorder-windows.zip`: Windows desktop build bundle
- `app-release.apk`: Android APK build

## Windows
1. Download `lecture-recorder-windows.zip`.
2. Extract the archive to a normal folder.
3. Run `lecture_recorder.exe`.

## Android
1. Download `app-release.apk`.
2. Transfer it to an Android device.
3. Allow installation from the chosen source if Android asks.
4. Open the app and grant the requested permissions for recording and notifications.

## Important Notes
- On Android, recordings are stored in `Music/Recordings/Lecture Recorder`.
- The app includes neutral sample course names only. Replace them in Settings if needed.
- The Windows build must stay together with its bundled DLL and data files.
